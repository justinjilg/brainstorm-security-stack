## What Is This?

A fully public, AI-agent-driven build of a complete MSP Security Stack:

| Component | Equivalent | What It Does |
|-----------|-----------|-------------|
| **CSPM Scanner** | Wiz | Cloud security posture management, misconfiguration detection |
| **EDR Agent** | CrowdStrike Falcon | Endpoint detection & response, threat hunting |
| **SIEM Engine** | Splunk/Elastic SIEM | Security event correlation, anomaly detection |
| **SOAR Playbooks** | Palo Alto XSOAR | Automated response, incident management |

**10 named AI agents** with persistent identities, domain expertise, and real budgets build this project in the open. Every commit, every PRD, every architecture decision, every code review is visible. Every LLM call routes through [BrainstormRouter](https://brainstormrouter.com) with full cost and routing transparency.

## Why?

The security stack is impressive. But **the real product being demonstrated is how it's built.**

- "This PRD was generated by Sage, routed through BrainstormRouter to GPT-5.4 ($0.06)"
- "This code review was a 3-agent consensus: Opus disagreed with Gemini, Sonnet broke the tie ($0.14)"
- "Total cost to build CSPM scanner: $4.23 across 847 routing decisions"

Every metric, every agent check-in, every artifact reinforces: [BrainstormRouter](https://brainstormrouter.com) makes AI development measurably better, cheaper, and more reliable.

## The Open Research Angle

This isn't a proprietary product. It's an **open research project**:

- Apache 2.0 — fork it, learn from it, contribute to it
- Every routing decision logged — "which model is best for security code review?"
- Every cost tracked — "what does it actually cost to build enterprise software with AI?"
- Every failure documented — "where do AI agents hallucinate in security-critical code?"
